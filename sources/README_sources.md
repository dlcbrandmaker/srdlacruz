# Fuentes internas usadas

Este sistema se construyó con base en documentos estratégicos y recursos visuales entregados por Jonathan de la Cruz Matías para su marca personal **sr.dlacruz**.

## Documentos analizados

- CV profesional de Jonathan de la Cruz M.
- Guía estratégica global de marketing, comunicación, marca y diseño.
- Guía estratégica global de marketing, comunicación, marca personal y psicología del consumidor.
- Guía estratégica integral para sr.dlacruz.
- Plan de acción personalizado Jonathan De La Cruz y Emprende Adentro.
- Documento de posicionamiento SRDLACRUZ.

## Nota de privacidad

El CV contiene datos personales de contacto. Para proteger la información privada en un repositorio público, este paquete no replica teléfonos ni datos sensibles. El sistema utiliza únicamente información profesional necesaria: perfil, áreas de experiencia, posicionamiento, propuesta de valor, tono, audiencia y dirección estratégica.
