# Prompt para OpenDesign — sr.dlacruz

Actúa como diseñador de marca senior, diseñador gráfico, diseñador UX/UI, estratega de contenido y arquitecto de sistemas visuales.

Trabaja sobre el repositorio:

https://github.com/dlcbrandmaker/srdlacruz.git

## Objetivo

Crear un sistema de diseño visual completo para la marca personal **sr.dlacruz / Jonathan de la Cruz M.**, usando todos los documentos, logos, retratos, tokens, prompts y referencias cargadas en el repositorio.

Antes de diseñar, lee completo:

- `README.md`
- `docs/`
- `assets/`
- `tokens/`
- `styles/`
- `prompts/`
- `sources/`

## Dirección de marca

La marca debe comunicar:

- Marketing real sin humo.
- Diseño con intención.
- Estrategia digital.
- Emprendimiento práctico.
- Autoridad visual.
- Claridad y criterio.
- Experiencia profesional.
- Cercanía con emprendedores latinos.

## Estética

Usar un sistema sobrio, premium y estratégico:

- Negro profundo.
- Gris grafito.
- Azul profundo.
- Azul eléctrico.
- Rojo de acento.
- Blanco.
- Retratos profesionales.
- Tipografía sans serif fuerte.
- Composición limpia.

## Fases

1. Auditar repositorio.
2. Validar assets.
3. Organizar sistema visual.
4. Crear tokens.
5. Crear componentes.
6. Crear plantillas 4:5, 9:16, 1:1 y 16:9.
7. Crear sistema para carruseles.
8. Crear sistema para reels.
9. Crear sistema para sitio/portafolio.
10. Documentar y validar.

## Reglas

- No alterar logos.
- No deformar retratos.
- No publicar teléfonos o datos sensibles.
- No usar estética genérica.
- No crear promesas exageradas.
- No usar frases vacías.
- Mantener tono directo y profesional.

## Entregables

- Diagnóstico del repositorio.
- Tokens visuales.
- Plantillas.
- Componentes.
- Guía de uso.
- Prompts mejorados.
- Checklist final.
