# Sistema Visual — sr.dlacruz

## Dirección visual

La marca debe combinar:

- Autoridad ejecutiva.
- Estética estratégica.
- Claridad educativa.
- Azul como confianza.
- Rojo como energía.
- Negro como poder y sobriedad.

## Formatos

### Instagram 4:5

- 1080 × 1350 px.
- Titular grande.
- 1 idea principal por slide.
- Fondo oscuro o claro según tema.
- Logo pequeño al cierre.

### Stories / Reels 9:16

- 1080 × 1920 px.
- Titulares breves.
- Subtítulos grandes.
- Espacio seguro superior e inferior.
- CTA al final.

### YouTube / video horizontal 16:9

- 1920 × 1080 px.
- Imagen personal o fondo conceptual.
- Texto máximo 5 palabras para miniatura.
- Contraste alto.

### Perfil / avatar

- Rostro claro.
- Fondo neutro.
- Evitar textos pequeños.
- Usar círculo de marca azul/rojo solo si no compite con el rostro.

## Componentes

- Hero con retrato.
- Card de consejo.
- Card de error común.
- Slide de framework.
- Slide de CTA.
- Bloque de frase.
- Portada de carrusel.
- Miniatura de reel.
- Plantilla de caso real.
- Plantilla de antes/después.
